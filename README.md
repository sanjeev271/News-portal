# 📰 News Portal

> A modern, full-stack news publishing platform with real-time updates, live broadcasting, and a powerful content management system.

News Portal is a production-ready **MERN Stack** application designed for digital media organizations, publishers, and content creators. It combines real-time news delivery, interactive reader engagement, and professional publishing tools into a single platform.

---

## 🚀 Key Highlights

✨ Real-time news publishing with Socket.io

📡 Live TV broadcasting using WebRTC and external streaming sources

📰 Rich article management with media uploads and SEO support

🔍 Advanced search and category filtering

❤️ Reader engagement through comments, likes, and bookmarks

🌙 Dark mode and multilingual support

📊 Admin analytics and content management dashboard

🔐 Secure JWT-based authentication and role management

---
